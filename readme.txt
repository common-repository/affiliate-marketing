== Description ==

Плагин интегрирует ваш сайт с платформой партнёрского маркетинга Sdelka.biz. 
С помощью которой вы можете создать свою партнёрскую программу, отслеживать партнёрские продажи или действия по реферальным ссылкам, а так же выплачивать партнёрам вознаграждения.

Плагин установит на ваш сайт код трекера для отслеживания партнёрских действий по реферальным ссылкам.
И создаст виджет для отображения ссылки на условия вашей партнёрской программы и получения реферальных ссылок.
